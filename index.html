Introdução

Bem-vindo à documentação do BR Framework.

O que é o BRFrame a final de contas?

É um framework totalmente de PHP onde a função principal é facilitar o uso de codigos

Ele é um framework privado onde tem direitos exclusivamente para a Brazil Arc

Desde comandos PHP simplificados onde você pode sair de codigos com mais de 50 linhas com um simples comando

Tambem incluindo arquivos completos de html ja com logicas PHP

Por exemplo: Tela de Login, Tela de Registro e muitos outros

Utilizado pelo prefixo: br()->comando

Onde o "comando" deve ser alterado pelos comandos disponibilizados pelo framework

Quais são os comandos e como utilizar?

Veja a categoria de comandos após a categoria de instalação para saber mais




Instalação

Para instalar o BR Framework, siga os passos abaixo:

Primeiro tenha em sua maquina o GIT

Você pode instalar o GIT gratuitamente pelo link: Git

Após a instalação execute o comando abaixo em seu diretorio do seu projeto para estar incluindo o framework

git clone https://github.com/kn1zone/BRFramework.git

Após isso o framework ja se encontra no seu diretorio

Para incluir as funcionalidades em suas paginas inclua o seguinte comando nas paginas .php

require 'framework-br/src/BR_Framework.php';

Quando incluido o framework estará funcionando 100% em sua pagina




Comandos Basicos

Função de conexão com banco de dados:

br()->conexaobd();

Cria uma conexão com banco de dados que pode ser configurado na pasta config

Função de iniciar sessão:

br()->sessionstart();

Cria uma sessão para o cliente que estiver utilizando o sistema

Função de VarDump da sessão:

br()->vardumpsession();

Cria um vardump direto para a sessão aberta

Função de coletar ip cliente:

br()->ip();

Coleta e exibe o ip do cliente

Função de coletar navegador do cliente:

br()->browser();

Coleta e exibe o navegador do cliente

Função de coletar localização do cliente:

br()->location();

Coleta e exibe a localização do cliente

Função de acabar com a session do cliente:

br()->sessionkill();

Coleta a session e destroi ela por completo

Função de logout:

br()->logout();

Faz o logout completo destruindo todos os vestigios

Função de redirecionar para url:

br()->redirect("https://www.exemplo.com");

Faz o redirecionamento do cliente para o url definido

Função de validar se a variavel está vazia ou preenchida:

br()->validatevar($valor);

Faz a validação se a variavel está vazia ou preenchida

Função de verificar se o site esta alocado em HTTPS:

br()->is_https();

Faz a verificação se o site esta alocado em HTTPS

Função de verificar se o site esta alocado em HTTP:

br()->is_http();

Faz a verificação se o site esta alocado em HTTP




Comandos Avançados

Função de autenticar usario:

br()->authuser();

Faz a verificação se o user fez o login normalmente

Função de gerar um UIA:

br()->uid();

Faz a geração de um UID

Função de verificar o quanto de memoria o script consome:

br()->memory();

Faz a verificação o quanto de memoria o script consome

Função de validar o email se é valido:

br()->validateemail();

Faz a validação do email se é valido

Função de gerar um codigo criptografado de uma senha:

br()->encryptpass("teste");

Faz o retorno de um codigo criptografado de uma senha

Função de verificar se a senha bate com o codigo criptografado:

br()->verifypass("teste", "$2y$10\$PB0vFMOWS/jWGhXu0PA1YepuV/ywYWeF3V//ZyHK6mwhh4ZynNtY2")

Faz a verificação se a senha bate com o codigo criptografado

Função de pesquisar dados na DB:

br()->searchdb("valor");

Faz uma pesquisa na DB

Função de criar conexão com SQL com 100% de segurança:

br()->prepare_sql_noinject("value")

Faz a conexão mais segura o possivel

Função de deletar um elemento de uma tabela:

br()->db_delete("tabela", "elemento")

Faz um delete um elemento de uma tabela

Função de inserir um elemento de uma tabela:

br()->db_insert("tabela", "dados")

Faz um inserir um elemento de uma tabela

Função de update um elemento de uma tabela:

br()->db_update("tabela", "dados", "condição")

Faz um update um elemento de uma tabela

Função de escapar um elemento html:

br()->escape_html("string");

Faz um escape de um elemento html

Função de criar um cookie por tempo limitado:

br()->secure_cookie("$nome_cookie", "$valor_cookie", "$tempo_expiracao");

Faz um cookie por tempo limitado

Função de criar um cache por tempo limitado:

br()->page_cache($chave, $conteudo, $tempo_expiracao);

Faz um cache que fica armazenado na pasta do seu servidor para ser reutilizado
